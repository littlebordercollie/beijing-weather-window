## 数据边界

- 18 个台站的站号、坐标和海拔来自北京市地方标准 DB11/T 1643—2023。
- 卡片实时数值是 Open-Meteo 站旁格点，不是台站原始观测；界面逐卡标注。
- 北京市水务局当前返回 121 个日雨量站点行，但响应没有经纬度，因此不进入附近距离计算。
- 五点探针落入不足 3 个独立模型网格时，程序拒绝推断雨带方向。
- 页面是出行辅助，不是官方气象预报；强对流天气以官方预警为准。

## 浏览器核验说明

本地生产页面完成了桌面、手机、地图、列表、半径切换和实时数据渲染检查。Codex 内置浏览器直接导航 GitHub Pages 时遇到 `ERR_CONNECTION_RESET`，因此公网部署本身使用以下独立证据验收：Pages 构建状态、HTTPS 200、首页/列表/脚本资源响应，以及公网文件与已完成视觉验收的本地生产文件逐字节哈希一致。没有把浏览器失败写成访问成功。
